Description du logiciel


STUDENITY qui est la contraction des deux mots student (étudiant) et community (communauté) est une application web d'entraide basée sur le partage et conçue pour les étudiants.

Elle a pour but de leur permettre d’acquérir des équipements (livres, équipements informatiques...) à des prix très abordables, de programmer des sessions de révision entre étudiants et d'avoir un espace de discussion.

Les principaux objectifs sont de favoriser le recyclage et de réduire la surconsommation par l’échange de biens et également de satisfaire de nombreux de leurs besoins, que ce soit financier, éducatif et sociaux.

Liens pour télécharger et installer


Téléchargements


Pour télécharger le site web au niveau local, vous trouverez les url sur GitHub à l'adresses suivante: https://github.com/AideEtudiants.

Le front end se trouve dans le repository: https://github.com/AideEtudiants/ProjetAngular et vous y trouverez tous les fichiers necessaires pour lancer le projet Angular.
Le back end se trouve dans le repository https://github.com/AideEtudiants/ProjetSpringBoot et contient les fichiers necessaires pour lancer le projet SpringBoot.
Il faut ensuite télécharger l'ensemble des fichiers contenus dans le front et ceux du back et les ouvrir dans l'IDE de votre choix, dans des fenetres differentes. De ce fait vous avez deux fenêtres ouvertes dans votre IDE contenant respectivement le front et le back.

La base de données se trouve dans le fichier: https://github.com/AideEtudiants/Documents-complementaires-et-README/blob/main/bdd.sql.
Le télechargement de Database Oracle et SQLDevelopper est nécessaire pour avoir accès à la base de données.


Installations


Vous devez créer une connexion Oracle et ajouter cette connexion sur SQLDevelopper. Ensuite il suffit d'ajouter notre fichier contenant notre base de données.

Lorsque vous clonnez le projet front, tapez "npm install" afin d'installer les dépendances nécessaires de Angular et par la suite "npm run build" afin de compiler et lancer le serveur avec la ligne de commande "ng serve". Si la commande ng n’est pas reconnue sur le terminal, ouvrez PowerShell en tant qu'administrateur et entrez "Set-ExecutionPolicy ExecutionPolicy Unrestricted".

Afin de compiler le projet Angular sans erreur, il faudra ajouter les materiaux de design angular avec cette commande: "ng add @angular/material".
Et pour ajouter la bibiothèque qui gère les teast, les commandes suivantes: "npm install ngx-toastr --save" et "npm install @angular/animations --save".

Clonner le projet SpringBoot sur votre IDE en s'assurant que votre IDE a bien un compiler Java.
Vous devez modifiez le username et le password dans le fichier "src/main/resources/application.properties" et ajoutez vos identifiant oracle.
Lancer le projet SpringBoot avec le bouton run sur votre IDE. Ensuite entrer cette url http://localhost:4200 afin d'afficher le résultat de l'API
